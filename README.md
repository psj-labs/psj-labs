### 🎯 Goal · Mission

> “To become a security professional who views IT systems as a whole, not as isolated pieces of technology.”

- Build a deep understanding of how <b>web services, networks, and systems are interconnected</b>  
- Learn from both offensive and defensive perspectives  
- Develop <b>practical, real-world security skills</b> applicable to live environments

---

### 🧭 Direction · Study Focus

- 🌐 **Web Hacking**  
  - Analyze vulnerability patterns such as input validation, authentication & session handling, and access control  
  - Focus on understanding <b>why vulnerabilities occur</b> by examining source code and HTTP request/response flows

- 🛜 **Network Security**  
  - Study protocol behavior including TCP/IP, DNS, and HTTP  
  - Practice identifying security issues directly from real network traffic at the packet level

- 🧩 **Comprehensive IT Security**  
  - Avoid viewing web, network, and system security as separate fields  
  - Strive to understand security as <b>one interconnected ecosystem</b>

---

### 🔐 Mindset · Security Philosophy

> “Trust nothing. Verify everything. Go deeper until the truth is clear.”

- Prioritize understanding <b>internal system behavior</b> over surface-level observations  
- Never consider knowledge complete until it has been <b>personally tested, reproduced, and validated</b>  
- Study offensive techniques with one purpose: <b>to design stronger and more effective defenses</b>

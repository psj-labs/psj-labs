<h2>웹 해킹 & 네트워크 보안 · 종합 IT 보안 중심</h2>

<p>
웹 해킹과 네트워크 지식을 출발점으로, 실습을 통해 종합적인 IT 보안을 학습합니다.
</p>

<ul>
  <li>공격 흐름을 직접 재현하고, 그 원인과 영향을 분석하는 방식으로 학습합니다.</li>
  <li>개별 기술이 아닌 <b>"시스템 전체를 바라보는 보안 관점"</b>을 기르는 데 집중합니다.</li>
</ul>

---

<h3>[ PoC(Proof of Concept) 안내 ]</h3>
<ul>
  <li>모든 리포지토리에는 실습 내용을 정리한 블로그 형태의 가독성 높은 PoC 문서가 포함되어 있습니다.</li>
  <li>PoC는 한국어로 작성되며 <b>원인 분석 · 재현 과정 · 보안적 의미</b>를 중심으로 기록합니다.</li>
  <li>각 PoC는 실제 공격 흐름을 기준으로 작성합니다.</li>
</ul>

---

<h3>[ 학습 목표 ]</h3>

<ul>
  <li>웹, 네트워크 등 종합적인 IT 보안측면에 대한 이해</li>
  <li>공격자 관점과 방어자 관점을 모두 고려한 분석</li>
</ul>

---

<h3>[지향하는 방향]</h3>
<ul>
 <li>현실적인 위협 시나리오를 이해하고 그에 맞는 대응 방안을 설계할 수 있는 보안 전문가를 지향합니다.</li>
 </ul>

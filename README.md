## 웹 해킹 & 네트워크 보안 · 종합 IT 보안 중심

- 웹 해킹과 네트워크 지식을 출발점으로, 실습을 통해 종합적인 IT 보안을 학습합니다.
- 공격 흐름을 직접 재현하고, 그 원인과 영향을 분석하는 방식으로 학습합니다.
- 개별 기술이 아닌 시스템 전체를 바라보는 보안 관점을 기르는 데 집중합니다.

---

## PoC 안내

- 모든 리포지토리에는 실습 내용을 정리한 블로그 형태의 가독성 높은 PoC 문서가 포함되어 있습니다.
- PoC는 한국어로 작성되며 재현 과정 및 실습과정 또는 개념중심으로 기록합니다.


---

## 학습 목표

- 웹, 네트워크, 리눅스 공부를 시작으로 종합적인 IT 보안측면에 대한 이해
- 공격자 관점과 방어자 관점을 모두 고려한 학습

## 지향하는 방향
- 위협 시나리오를 이해하고 그에 맞는 대응 방안을 설계할 수 있는 보안 전문가를 지향합니다.
